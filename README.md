# QA Portfolio – Vinicius Pedro

Este repositório contém exemplos práticos de testes de software realizados com foco em QA Manual, incluindo criação de casos de teste, identificação e documentação de bugs e testes de API.

## Habilidades demonstradas

- Testes manuais (funcionais e exploratórios)
- Criação de cenários e casos de teste
- Identificação e documentação de bugs
- Testes de API com Postman
- Noções de automação (Cypress)
- Uso de ferramentas como Jira (conceitual)

## Sistemas testados

- SauceDemo (https://www.saucedemo.com/)
- APIs públicas (JSONPlaceholder)

## Estrutura

- `test-cases/` → Casos de teste
- `bug-reports/` → Bugs documentados
- `api-tests/` → Testes de API
- `evidences/` → Prints e evidências

## Objetivo

Demonstrar habilidades práticas em  Analista de Testes / QA Junior.
